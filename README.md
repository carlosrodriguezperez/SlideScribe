# SlideScribe

**SlideScribe** is an AI-powered academic suite designed to transform lecture slides into structured, mathematically rigorous Markdown notes and active studying tools.

---

## 🗺️ Project Structure

This repository is structured as a monorepo containing:

* **[cli/](./cli):** The core command-line tool that transcribes slide PDFs, salvages math formulas into LaTeX, and generates explanations.
* **Future Projects (Planned):** A web interface with RAG chat and custom exam generators.

For installation, usage, and detailed features of the CLI tool, please refer directly to the **[SlideScribe-CLI README](./cli/README.md)**.
